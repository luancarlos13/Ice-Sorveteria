## 🍦 Landing Page - Sorveteria

## 📌 Sobre o Projeto

Esta landing page foi desenvolvida com foco no segmento de sorveterias, com o objetivo de apresentar produtos de forma atrativa e estimular o desejo do usuário através de um design visual envolvente.

O layout foi pensado para destacar sabores, promoções e identidade da marca, conduzindo o visitante até a ação principal (CTA), como realizar um pedido ou entrar em contato.

## 🎯 Objetivo

A página tem como principal objetivo atrair e converter visitantes em clientes, utilizando elementos visuais chamativos e uma navegação simples e intuitiva.

Landing pages são projetadas justamente para guiar o usuário a uma única ação, reduzindo distrações e aumentando as chances de conversão .

## 🧩 Estrutura da Página

A landing page segue uma estrutura moderna e eficiente:

- 🍦 Header – Identidade visual e navegação clara
- 🎯 Seção Principal (Hero) – Destaque para os produtos e chamada atrativa
- 🍨 Cardápio / Sabores – Apresentação dos sorvetes disponíveis
- ⭐ Diferenciais – Qualidade, variedade e experiência
- 💬 Depoimentos – Prova social e confiança
- 📞 Contato / CTA – Chamada para ação estratégica

Cada seção foi pensada para criar uma experiência agradável e levar o usuário até a decisão de compra.

## 🎨 Design e Experiência

O design utiliza cores vibrantes (roxo e amarelo), elementos visuais atrativos e imagens que despertam o apetite, criando uma experiência envolvente.

Além disso, a página é:

- 📱 Responsiva – Adaptada para mobile, tablet e desktop
- ⚡ Leve e rápida – Foco na performance
- 🎯 Direta – Estrutura otimizada para conversão
- 😍 Visualmente atrativa – Estilo moderno e chamativo

## 🚀 Tecnologias Utilizadas

- HTML5 🟧
- CSS3 🟦
- JavaScript 🟨

## 💡 Considerações

Este projeto demonstra como uma landing page bem construída pode ser uma poderosa ferramenta de marketing digital, funcionando como uma vitrine digital focada em conversão e experiência do usuário .
